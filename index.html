Mis proyectos

Juego web "¿Cuál es más caro?"

Juego basado en HTML y JS, disponible a través de Internet.

El objetivo del juego es hacer la mayor racha posible acertando cúal de los dos objetos mostrados es más caro. Utiliza HTML, CSS y JavaScript. Al crearlo utilicé Bootstrap para facilitar alguna parte del diseño, aunque con esta página que estás leyendo ahora he aprendido a hacer páginas web responsivas sin Bootstrap.

Gracias a esta página aprendí a programar en JavaScript, ya que la puntuación, el seleccionar dos objetos y sus imágenes, sus precios, y muchas más cosas, funcionan con JavaScript. Se puede acceder a la página haciendo click en la imagen de arriba.

Bot de Discord: "El taquillero"

Bot para el servicio de chat Discord.

Este bot, basado en Python, utiliza dos librerías. Una le permite conectarse a Discord y usar sus funciones, y otra extraer información de IMDb. Mediante dos comandos, puede buscar información de la película o serie que solicites y enviártela, resumidamente en un mensaje de texto.

Este fue el primer bot de Discord que hice y la primera vez que usé estas dos librerías, aunque ya tenía conocimientos previos de Python y de otras librerías. Se puede añadir el bot a un servidor de Discord haciendo click en la imagen de arriba.

Generador de malas traducciones

Programa basado en Python que, aunque no tiene ninguna función realmente práctica, me parece interesante.

Usa una librería para conectarse al traductor de Google, le pide al usuario indicar si va a usar el programa en inglés o español, le pide un texto al usuario y pasa este texto por muchos idiomas y lo devuelve al idioma original, para conseguir una traducción cómicamente mala.

Este programa me gusta porque fue el primero que hice usando una librería en Python, y me costó algo hacerlo, pero también aprendí mucho gracias a él. Se puede acceder al código en Github haciendo click en la imagen de arriba.

Juego web "Filmle"

Juego basado en HTML y JS, disponible a través de Internet.

El objetivo es adivinar una película, elegida al azar de una lista. Para adivinarla, debes ir intentando adivinar otra información de la película que te facilite la tarea: el año (del cuál se te dirá también si acertaste la década o el siglo), el género, el director.

Poco a poco con estas pistas puedes ir acercándote hasta, con un poco de suerte, adivinar la película.

También puedes hacer click en los títulos de cada categoría (año, director, género) para descubrir directamente cuál es cada uno, gastando unos intentos.

Desarrollé esta página con el objetivo de aprender a programar formularios, y practicar un poco HTML y CSS. Se puede acceder haciendo click en la imagen de arriba.

Página web "LCS"

Página web con backend en NodeJS.

Es una página sobre un equipo de carreras virtual que hice con unos amigos para pasar el rato. La página tiene un backend usando NodeJS, bastante simple pero funcional. Concretamente, al enviar una consulta en la página del formulario, esta se envía al servidor (la página está alojada en un servidor mío). Una vez llega al servidor, la consulta se procesa para finalmente guardarse en un archivo de texto en el servidor.

Es un backend muy simple y no especialmente seguro, pero el propósito de la página era aprender a hacer una página con backend, ya que, aunque trabajé con lenguajes de backend, como Python, nunca había creado una página con backend.

También, en cuanto al frontend, es una de las páginas más elaboradas que he desarrollado, con modo oscuro, diseño responsivo, menú desplegable en móvil... Se puede acceder haciendo click en la página de arriba.
